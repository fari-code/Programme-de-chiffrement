# 🔐 Programme de Chiffrement

Ce projet est une implémentation en Python de plusieurs algorithmes de chiffrement classiques utilisés en cryptographie.

## 🚀 Fonctionnalités

- 🔑 Chiffrement de César
- 🔁 ROT13
- 🔐 Chiffrement de Vigenère
- 🔤 Chiffrement par substitution
- 🔓 Possibilité de déchiffrement

## 🧠 Objectif

Ce projet a pour but de :
- Comprendre les bases de la cryptographie
- Manipuler les chaînes de caractères en Python
- Appliquer des algorithmes classiques
- Renforcer les compétences en programmation

## 🛠️ Technologies utilisées

- Python
